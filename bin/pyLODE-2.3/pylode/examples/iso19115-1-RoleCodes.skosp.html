Description

A SKOS version of the ISO 19115-1 CI_RoleCode Codelist. These roles are the roles of parties (Agents) with respect to datasets (Entities). They are taken from the international standard ISO19115-1:2014 Geographic information - Metadata - Part 1: Fundamentals. From ISO19115-1:2014, Table B.16.1 - Responsible party information on page 91 lists properties of the CI_Responsibility class which is 'information about the party and their role'. The 'role' property, indicating a value from the CI_RoleCode codelist - the source of this vocabulary - is described as the 'function performed by the responsible party'.
