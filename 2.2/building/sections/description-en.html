Types of Room

In the current version, only types of Room is defined in the Building ontology. A user can modify or create a new a list of needed room types. Other types of controlled vocabularies needed to describe building parts are recommended to place in the Building ontotlogy.

The list of different types of Room:s is an example of a list of predefined instances that constitues of a controlled vocabulary.
