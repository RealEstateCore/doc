Description

This REC module covers the data model used by government agency Lantmäteriet, the Swedish National Land Survey, in its Fastighetsregister, Swedish Real Property Register (SRPR)

Description

City name.

Note that object type 90A field 15 ("ADRTYP") holds the key to interpretation of this field; value = 3 (~byadressområde) implies a city name (bynamn).

Also note that city name may be different to the postal location.
